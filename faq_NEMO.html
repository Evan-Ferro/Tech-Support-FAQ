Why isn't the simulator working on my Ipad?

Version 0.0

If a user is unable to access a certain Simulator on NEMO this is mostly because of the device they are using.

Some of the older Simulators such as Handheld VHF, VHF & MFHF may allow users to access through ipad or other mobile devices.

But as for the more recent simulators like Radar, Sextant etc... they should not be able to use them.
• Please advise to the user that they should switch to a PC if possible and apologize for the inconvenience

‎‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎‏‏‎ All of your courses will show up in their respective categories here.

My course module is not being marked as “Complete” after I’ve finished it, what do I do?

Version 1.0

• If you have completed a module within your course and it is still showing “In Progress”,
‎‏‏‎ ‎‏‏‎ ‎‏‏‎please make sure you are using Google Chrome if possible as this is
‎‏‏‎ ‎‏‏‎ ‎‏‏‎the preferred web browser for all NEMO courses.

• Whenever you exit a course module,
‎‏‏‎ ‎‏‏‎ ‎‏‏‎ make sure you are clicking the yellow [X] in the top-right corner.

• If you have refreshed the page and followed these tips and it is still not being marked as
‏‏‎ ‎‏‏‎ “Complete”, please email Tech Support or call our office to be assisted.
