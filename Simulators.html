Why isn't the simulator working on my Ipad?

Version 0.0

If a user is unable to access a certain Simulator on NEMO this is mostly because of the device they are using.

Some of the older Simulators such as Handheld VHF, VHF & MFHF may allow users to access through ipad or other mobile devices.

But as for the more recent simulators like Radar, Sextant etc... they should not be able to use them.
Please advise to the user that they should switch to a PC if possible and apologize for the inconvenience
